5G NR Basics

Fundamental concepts of 5G New Radio technology and architecture

1. What is 5G NR?

5G New Radio (NR) is the global standard for a unified, more capable 5G wireless air interface. It provides:

  • Higher data rates (up to 20 Gbps)
  • Lower latency (< 1ms)
  • Massive connectivity (1 million devices/km²)
  • Enhanced reliability and security
Key Innovation: 5G NR uses flexible numerology and frame structure to support diverse use cases from enhanced mobile broadband (eMBB) to ultra-reliable low-latency communications (URLLC).

2. 5G NR vs LTE Comparison

Feature LTE 5G NR
Peak Data Rate ~1 Gbps Up to 20 Gbps
Latency ~10-20 ms < 1 ms
Subcarrier Spacing 15 kHz 15, 30, 60, 120 kHz
Frame Structure Fixed 10ms frames Flexible numerology
Bandwidth Up to 20 MHz Up to 400 MHz
MIMO Up to 8x8 Massive MIMO (64x64+)

Subcarrier Spacing = 15 × 2^μ kHz
(where μ = 0, 1, 2, 3, 4)
μ = 0 (15 kHz)

eMBB applications, wide coverage, LTE compatibility

μ = 1 (30 kHz)

Balanced performance for various use cases

μ = 2 (60 kHz)

Low latency applications, URLLC

μ = 3 (120 kHz)

High-frequency mmWave bands

5. Key Technologies

5G NR Core Technologies:

  • OFDM: Orthogonal Frequency Division Multiplexing for efficient spectrum use
  • Massive MIMO: Multiple antenna arrays for beamforming and spatial multiplexing
  • Beamforming: Directional signal transmission for improved coverage and capacity
  • Network Slicing: Virtual network segments for different service requirements
  • Carrier Aggregation: Combining multiple frequency bands for higher throughput

6. Use Cases and Service Types

eMBB (Enhanced Mobile Broadband)

High data rate applications: 4K/8K video streaming, AR/VR, high-speed internet

URLLC (Ultra-Reliable Low Latency)

Mission-critical applications: autonomous vehicles, remote surgery, industrial automation

mMTC (Massive Machine Type Communications)

IoT applications: smart cities, agriculture, asset tracking
