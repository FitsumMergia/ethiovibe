LTE Frame Structure Basics

Fundamental concepts of LTE frame structures, timing, and frequency domains

1. LTE Frame Structure Overview

LTE uses a flexible frame structure that supports both Frequency Division Duplex (FDD) and Time Division Duplex (TDD) modes. The basic unit is the radio frame with a duration of 10ms.

[LTE Frame Structure Diagram]
10ms Radio Frame → 10 Subframes → 20 Slots → 140 Symbols

2. Key Parameters

Parameter Value Description
Frame Duration 10 ms Length of one radio frame
Subframe Duration 1 ms Each frame contains 10 subframes
Slot Duration 0.5 ms Each subframe contains 2 slots
Symbol Duration ~71.4 μs Each slot contains 7 symbols (normal CP)

3. Frequency Domain

Key Points:
  • Channel bandwidth ranges from 1.4 MHz to 20 MHz
  • Subcarrier spacing: 15 kHz
  • Number of subcarriers depends on bandwidth
  • Resource Block (RB) = 12 subcarriers × 7 symbols

4. Time-Frequency Resource Grid

Resource Element (RE) = 1 subcarrier × 1 symbol

The time-frequency resource grid is the basic framework for LTE resource allocation.

5. Cyclic Prefix

LTE uses Cyclic Prefix (CP) to combat multipath interference:

  • Normal CP: 4.7 μs for first symbol, 5.2 μs for others
  • Extended CP: 16.7 μs for all symbols
