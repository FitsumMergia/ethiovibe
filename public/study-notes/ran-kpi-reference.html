RAN KPI Reference Guide

Comprehensive reference for Radio Access Network performance indicators

1. Accessibility KPIs

Measuring network availability and connection success rates
RRC Setup Success Rate (RRC SSR)
RRC SSR = (RRC Setup Complete) / (RRC Setup Attempts) × 100%
Target: > 98%
Measures the success rate of RRC connection establishments. Critical for user experience.
E-RAB Setup Success Rate (E-RAB SSR)
E-RAB SSR = (E-RAB Setup Success) / (E-RAB Setup Attempts) × 100%
Target: > 97%
Success rate of bearer establishment after RRC connection.

2. Retainability KPIs

Measuring call drop rates and connection stability
Call Drop Rate (CDR)
CDR = (Abnormal Releases) / (Total Calls) × 100%
Target: < 2%
Percentage of calls that drop unexpectedly during conversation.
Handover Success Rate (HOSR)
HOSR = (Successful Handovers) / (Handover Attempts) × 100%
Target: > 98%
Success rate of handovers between cells during mobility.

3. Mobility KPIs

Measuring handover performance and mobility management
Intra-LTE Handover Success Rate
Intra-LTE HOSR = (Successful Intra-LTE HO) / (Intra-LTE HO Attempts) × 100%
Target: > 98%
Success rate of handovers within the LTE network.
Inter-RAT Handover Success Rate
IRAT HOSR = (Successful IRAT HO) / (IRAT HO Attempts) × 100%
Target: > 95%
Success rate of handovers between different radio access technologies.

4. Throughput KPIs

Measuring data transfer performance
Average User Throughput
Avg Throughput = Total Data Volume / Total Connection Time
Target: Varies by bandwidth
Average data rate experienced by users during active connections.
Cell Throughput
Cell Throughput = Total Cell Data Volume / Measurement Period
Target: Based on cell capacity
Total data throughput handled by a cell over time.

5. Coverage KPIs

Measuring network coverage and signal quality
KPI Name Formula Target Description
Reference Signal Received Power (RSRP) Average RSRP measurement > -100 dBm Signal strength indicator
Reference Signal Received Quality (RSRQ) Average RSRQ measurement > -10 dB Signal quality indicator
Signal-to-Interference-plus-Noise Ratio (SINR) Average SINR measurement > 10 dB Signal quality relative to interference

6. Capacity KPIs

Measuring network utilization and capacity
Cell Load
Cell Load = (Used PRB) / (Total PRB) × 100%
Target: < 80%
Percentage of Physical Resource Blocks utilized in the cell.
User Count per Cell
Average Connected Users = Total Connected Users / Number of Cells
Target: Based on cell capacity
Average number of simultaneously connected users per cell.
